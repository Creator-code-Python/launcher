=== TERMINAL LAUNCHER ===
Версия: 1.0
Автор: [ТВОЕ ИМЯ]

Описание:
Терминальный лаунчер для скачивания CFG, Skin и GUI файлов.
Поддерживает подменю для выбора конкретных файлов и автообновление через config.json.

---

СОДЕРЖАНИЕ ПРОЕКТА:

LauncherProject/
├── launcher.py       # Терминальный лаунчер для пользователей
├── uploader.py       # Админ-программа для добавления файлов
├── config.json       # Конфигурация всех файлов
├── install.bat       # Установщик для пользователей (опционально)
└── README.txt        # Эта инструкция

---

1️⃣ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

1. Скачайте архив LauncherPack.zip
2. Распакуйте в любую папку
3. Запустите launcher.exe
4. Главное меню:
   1. Скачать CFG      → Подменю с конкретными CFG файлами
   2. Скачать Skin     → Подменю с конкретными Skin файлами
   3. Скачать GUI      → Подменю с GUI файлами
   9. Скачать всё      → Скачивает все доступные файлы
   0. Выход            → Закрывает лаунчер
5. Файлы автоматически сохраняются в папку:
   C:\Users\<Ваше_имя>\MyLauncher

---

2️⃣ ДЛЯ АДМИНА (добавление новых файлов)

1. Запустите uploader.py
2. Выберите:
   1. Добавить файл → CFG или Skin
   2. Обновить GUI → изменить GUI лаунчера
   0. Выход
3. Введите:
   - Название файла
   - Имя файла (для сохранения)
   - Прямую ссылку (URL)
4. После добавления обновите config.json на сервере (GitHub raw)
5. Пользователи при запуске лаунчера автоматически увидят новые файлы

---

3️⃣ СОЗДАНИЕ .EXE (для пользователей без Python)

1. Установите PyInstaller:

   pip install pyinstaller

2. Перейдите в папку проекта с launcher.py
3. Соберите exe:

   pyinstaller --onefile --noconsole launcher.py

4. Готовый файл появится в папке dist/launcher.exe
5. Можно добавить иконку:

   pyinstaller --onefile --noconsole --icon=icon.ico launcher.py

---

4️⃣ СОВЕТЫ

- Все файлы (CFG, Skin, GUI) должны быть доступны по прямым ссылкам
- Поддерживается любое количество файлов, новые файлы добавляются через uploader.py
- Главное меню + подменю удобно разделяют категории файлов
- Для быстрого старта пользователей можно использовать install.bat
